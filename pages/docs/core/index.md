# Core Infrastructure

## Overview

The Core Infrastructure section documents the critical infrastructure services that form the foundation of the SeekerAI platform. These services must remain stable and reliable as they support all other applications and services.

## Services

### LiveKit Infrastructure

WebRTC infrastructure, room management, and real-time communication services.

[Learn more about LiveKit Infrastructure](livekit-infra/index.md)

### Nginx

Reverse proxy, SSL/TLS termination, and routing services.

[Learn more about Nginx](nginx/index.md)

### SIP Infrastructure

SIP proxy, registration keeper, and telephony integration services.

[Learn more about SIP Infrastructure](sip/index.md)
